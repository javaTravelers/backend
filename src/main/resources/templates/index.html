Java Travelers Rest API - 1.0

JavaTravelers Rest API foi desenvolvida para permitir a busca de passagens aéreas, bem como favoritar, e realizar a compra integrada com o PagSeguro. O projeto foi desenvolvido durante o treinamento realizado pela Gama para os New Joiners da Accenture.
